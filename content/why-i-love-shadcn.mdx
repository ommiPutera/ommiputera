Sebagian komponen dari **ommiputera.com** dibangun menggunakan [**Shadcn/UI**](https://ui.shadcn.com/). 

### Alasan utama

Inilah alasan utama saya:

> Shadcn/UI memberikan saya fleksibilitas tinggi dalam menyesuaikan komponen-komponennya dengan berbagai *design system*.

Jadi, apa artinya? Mari kita bahas..

### Kontrol penuh

Dengan Shadcn/UI, saya memiliki kontrol penuh atas kode setiap komponen UI yang saya gunakan. Ini berarti saya dapat:

- Menyesuaikan komponen secara mendetail untuk mencocokkan *design system* apa pun.
- Melakukan pengembangan lanjutan tanpa terikat pada batasan bawaan framework.
- Mengoptimalkan UI untuk memastikan performa tinggi, aksesibilitas, dan kemudahan dalam *maintainance*.

Berbeda dengan framework UI konvensional seperti [**Mantine**](https://mantine.dev/), [**Bootstrap**](https://getbootstrap.com/) yang mengharuskan kita untuk menyesuaikan UI desain berdasarkan aturan mereka dan terkadang sangat *pain* untuk memodifikasi komponennya.

### Modularitas

> Hanya menginstall komponen yang relevan pada projectX

Salah satu keunggulan terbesar Shadcn/UI adalah modularitasnya. Tidak seperti framework UI lain yang mengharuskan saya untuk menginstal seluruh paket komponen, meskipun tidak semuanya dibutuhkan, Shadcn/UI memungkinkan saya untuk hanya menginstal komponen yang relevan dengan proyek.

Pendekatan ini memberikan beberapa manfaat:

- Menghemat ruang penyimpanan karena Anda tidak perlu menginstal seluruh paket komponen.
- Meningkatkan efisiensi pengembangan dengan memungkinkan fokus pada fitur-fitur yang benar-benar penting.

Misalnya, untuk menambahkan komponen Input:

```sh
npx shadcn@latest add input
```

Dengan hanya menjalankan perintah di atas, saya sudah mendapatkan komponen Input yang aksesibel, modern, dan memiliki desain minimalis yang langsung siap digunakan.

### Kesimpulan 

Dengan fleksibilitas tinggi, kontrol penuh, dan pendekatan modular, Shadcn/UI telah menjadi salah satu alat favorit saya dalam membangun ommiputera.com. Hasilnya adalah sebuah aplikasi yang tidak hanya terlihat menarik, tetapi juga berjalan dengan baik, mudah diakses, dan gampang untuk dirawat di masa mendatang.
